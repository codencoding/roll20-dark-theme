# Roll 20 Dark Theme

A Firefox extension adjusting the css for some Roll20 pages to be a darker theme. 
<br>
Downloadable [here](https://codencoding.github.io/roll20-dark-theme/).
- roll20.net/forum
- roll20.net/campaigns
